# Netflix Data Analysis & Machine Learning

## Overview
This project analyzes Netflix Movies and TV Shows data using Data Analysis, Visualization, and Machine Learning techniques.  
The goal is to discover content trends, genre popularity, and build classification models.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- WordCloud

## Project Workflow
- Data Loading
- Data Cleaning
- Handling Missing Values
- Data Visualization
- Feature Encoding & Scaling
- Machine Learning Model Building
- Model Evaluation

## Visualizations Included
- Top Genres on Netflix
- Genre Distribution
- Genre Trends Over Years
- Movies vs TV Shows Comparison
- Country-wise Genre Analysis
- Genre Heatmap
- Word Cloud

## Machine Learning Models
| Model | Accuracy |
|---|---|
| Logistic Regression | 69.7% |
| KNN Classifier | 82.8% |

## Key Insights
- Movies are more common than TV Shows on Netflix.
- Drama and International genres are highly popular.
- Netflix content increased rapidly after 2018.
- KNN performed better than Logistic Regression.

## Conclusion
This project demonstrates how Data Analysis and Machine Learning can be used to gain insights from entertainment datasets and improve content classification.
